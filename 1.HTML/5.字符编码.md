# 5. 字符编码

## 5.1 字符编码相关概念
* 编码: 将字符转换为2进制码的过程。

* 解码: 将2进制码转换为字符的过程。

* 字符集: 编码和解码所采用的规则。

## 5.2 乱码问题
乱码问题: 如果编码和解码所采用的额字符集不同就会出现乱码问题。

## 5.3 常见字符集
* ASCII
* GB2312
* GBK
* UTF-8

## 5.4 设置html的字符集

head标签里面的meta标签来设置网页的字符集，避免出现乱码问题。

格式:
```html
<meta charset="utf-8">
```

如:
```html
<head>
    <meta charset="UTF-8">
</head>
```
