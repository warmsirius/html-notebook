# 12. 长度单位

* 像素
* 百分比
* `em`
* `rem`

## 12.1 像素

像素: 屏幕(显示器)实际上是由1个1个小点构成。

不同屏幕的像素大小是不同的，像素越小的屏幕，显示效果越清晰，所以，同样的200px在不同设备下显示效果不一样。


## 12.2 百分比

百分比: 将属性值设置为相对于父元素属性的百分比。


## 12.3 `em`
em: 相对于 自身元素 的字体大小来计算的。

em会根据字体大小改变而改变。

```text
1em = 1font-size
```

```css
.box3{
    font-size: 16px;
    width: 100px;
    height: 20em;
    background-color: green;
}
```


## 12.4 `rem`
rem: 相对于 根元素 字体大小来计算的。
