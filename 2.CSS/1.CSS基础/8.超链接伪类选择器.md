# 8. 超链接伪类选择器

超链接标签的状态:
* 1.没有访问过的链接
* 2.访问过的链接
* 3.鼠标悬浮
* 4.鼠标点击


## 8.1 超链接伪类

* `:link`: 表示没访问过的链接(正常的链接)
* `:visited`: 访问过的链接
  
> `说明`: 由于隐私的原因，所以 `:visited` 伪类只能修改链接的颜色，其余字体等都不能修改。

* `:hover`: 表示鼠标移入的状态
* `:active`: 表示鼠标正在点击的状态

`注意`: `:link`、`:visited` 是 a标签 特有的，`:hover` 和 `:active` 是所有元素都可以使用的伪类。
